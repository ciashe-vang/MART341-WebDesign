Valorant

Designed by Riot Games

Valorant is a PC first person-shooter designed to make players compete against each other in order to win the game. There are different game modes such as Unrated, Competitive, Spike Rush, and an interchanging rotation of Escalation and Replication. These game modes are suited for every player's play style.

There are different ranks for competitive so each game is unique to the player.
